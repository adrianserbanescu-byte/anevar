Teren si constructie

Suprafata terenului din actul cadastral. Baza pentru valoarea terenului in abordarea prin cost. Suprafata interioara efectiv folosibila a constructiei (fara pereti). Folosita in descrierea fizica si la verificari (Au nu poate depasi Acd). Suprafata totala construita pe toate nivelurile (cu pereti). Este numitorul pentru costul pe mp (CIN) si suprafata subiectului folosita la comparatie. Anul fata de care se calculeaza varsta constructiei. Varsta fiecarui element = an referinta − an punere in functiune (an_pif). Determina deprecierea. Valoarea estimata a terenului (din comparatie sau alta sursa). Se adauga la costul net al constructiei (CIN) pentru valoarea prin cost.

Metoda de evaluare

Care abordare determina valoarea finala. „Cost" = constructie prin cost de inlocuire net + teren. „Piata" = grila de comparatie. „Ponderata" = media celor doua.

Elemente de cost (un rand per element)

Fiecare element constructiv din catalogul IROVAL: infrastructura, structura, finisaje, instalatii, invelitoare. Format pe linie: element;cod;um;cantitate;cost_unitar;an_pif. Suma (cantitate × cost_unitar) = costul de inlocuire brut (CIB). „an_pif" = anul punerii in functiune (pentru varsta/depreciere).

Depreciere fizica (puncte varsta;fractie)

Punctele din tabelul de depreciere intre care se interpoleaza deprecierea fizica (Dfn), in functie de varsta cronologica ponderata (Vcp). Format pe linie: varsta;fractie (ex. 30;0.31 = la 30 ani, 31% depreciere).

Comparabile de piata (un rand: pret;suprafata)

Proprietati similare vandute/oferite, folosite la metoda „piata". Din pret si suprafata se calculeaza pretul unitar (pret/suprafata) si se selecteaza valoarea. Recomandat minimum 3 comparabile. Format pe linie: pret;suprafata.
